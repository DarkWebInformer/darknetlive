Dark.Fail Releases Tool for Checking Signed Messages

~1 min read | Published on 2020-01-21, tagged General-News, Tor using 98 words.
The “PGP Tool,” created by Dark.Fail, provides users of onion services with a quick way to verify the authenticity of signed messages from marketplaces, forums, news sites, and other popular services.
Dark.fail announced “Dark.Fail’s PGP Tool” on Twitter as well as on the homepage of Dark.Fail itself. To use the tool, you simply paste a signed message into the form and click submit. Invalid messages return “BAD signature from the public key provided!” and valid ones return “GOOD signature by [the identity associated with the key].”
Check the tool out here (the clearnet service) Dark.Fail/pgp and here (onion service) darkfailllnkf4vf.onion/pgp.
