HugBunter: Dread will Return Shortly

~2 min read | Published on 2020-01-26, tagged Dread using 409 words.
Dread will be back online in the near future, according to a status update posted by Dread’s administrator.
On the /r/dreadalert subreddit, HugBunter posted his (likely) final update before relaunching Dread. Text below.
Whilst I’d rather not give an exact timeframe to stick to, we’re down to some final setup and adjustments which shouldn’t take too much longer. Sorry that were not prepared to recover from such a circumstance so quickly, it was completely out of the realm of possibilities we had imagined and with the complex setup we put in place for the cluster, it is not something that can be set up over night. With us having to set this up from scratch again, whilst we have improved on it overall, this has lead to other issues occurring along the way which Paris is still working hard to resolve.

Didn’t want to keep everyone in the dark any longer, but work is on going and Dread will return with some small feature updates which I will complete this evening.

Thank you for the continued patience and support!
